효과 명령어 상세 (Effect Commands Detail)

1. 시각 효과 (Visual Effects)

fadein

사용 타입: onbgimage, oncharacter

설명: 객체가 투명에서 불투명으로 서서히 나타납니다.

{
    "type": "oncharacter",
    "Value01": "jung",
    "Value02": "normal",
    "Value03": "center",
    "Value04": "fadein",
    "Value05": 2.0  // 2초 동안 페이드인
}

fadeout

사용 타입: offcharacter

설명: 객체가 불투명에서 투명으로 서서히 사라집니다.

{
    "type": "offcharacter",
    "Value01": "jung",
    "Value04": "fadeout",
    "Value05": 1.5  // 1.5초 동안 페이드아웃
}

blind

사용 타입: onbgimage

설명: 블라인드 효과로 위에서 아래로 펼쳐지며 나타납니다.

{
    "type": "onbgimage",
    "Value01": "assets/images/bg/room.jpg",
    "Value02": "blind",
    "Value03": 1.5  // 1.5초 동안 블라인드 효과
}

shake

사용 타입: anieffect

설명: 객체가 좌우로 흔들리는 애니메이션 효과입니다.

{
    "type": "anieffect",
    "Value01": "jung",  // 또는 "all"
    "Value02": "shake",
    "Value03": 0.5  // 0.5초 동안 흔들림
}

2. 위치 명령어 (Position Commands)

center

사용 타입: oncharacter, move

설명: 객체를 화면 중앙에 배치합니다 (50%).

{
    "type": "oncharacter",
    "Value01": "jung",
    "Value02": "normal",
    "Value03": "center"  // 중앙 배치
}

left

사용 타입: oncharacter, move

설명: 객체를 화면 왼쪽에 배치합니다 (20%).

{
    "type": "move",
    "Value01": "jung",
    "Value02": "left",  // 왼쪽으로 이동
    "Value03": 1.0  // 1초 동안 이동
}

right

사용 타입: oncharacter, move

설명: 객체를 화면 오른쪽에 배치합니다 (80%).

{
    "type": "oncharacter",
    "Value01": "jung",
    "Value02": "normal",
    "Value03": "right"  // 오른쪽 배치
}

3. 기타 명령어 (Other Commands)

loop

사용 타입: onbgm

설명: 오디오를 반복 재생합니다.

{
    "type": "onbgm",
    "Value01": "assets/audio/bgm/main_theme.mp3",
    "Value02": "loop"  // 반복 재생
}

all

사용 타입: anieffect, transform

설명: 모든 객체 또는 모든 캐릭터에 효과를 적용합니다.

{
    "type": "anieffect",
    "Value01": "all",  // 모든 객체에 적용
    "Value02": "shake",
    "Value03": 1.0
}

개발자 도구 (Developer Tools)

Index Jump (인덱스 점프)

테스트를 위해 게임 도중 특정 인덱스로 즉시 이동할 수 있는 기능입니다.

  • 사용법: SETTING (Option) 클릭 -> Sound 탭 하단의 INDEX JUMP 칸에 이동할 인덱스 입력 -> 설정창 닫기 (또는 Enter)
  • 주의사항: 이 기능은 개발 및 테스트 전용이며, 배포 시에는 제거되거나 주석 처리되어야 합니다.

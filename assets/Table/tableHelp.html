Game Engine Type Reference
JSON Schema & Functionality Guide (v0.8.1)

Overview

This document details all supported type values for PLAY.JSON. Each event object strictly follows the schema:

{
    "Index": 100,          // Unique ID (Integer)
    "type": "string",      // Event Type
    "Value01": "...",      // Parameter 1
    "Value02": "...",      // Parameter 2
    "Value03": "...",      // Parameter 3
    "Value04": "...",      // Parameter 4
    "Value05": "..."       // Parameter 5
}

2. Detailed Examples

Background & Visuals

bgcolor: Smoothly transitions the background color.

{
    "Index": 100,
    "type": "bgcolor",
    "Value01": "#000000",
    "Value02": 2.0  // Transition takes 2 seconds
}

onbgimage: Fades in a background image. Supports multiple layers (higher number = front).

{
    "Index": 110,
    "type": "onbgimage",
    "Value01": "assets/images/bg/bg_room.jpg",
    "Value02": "fadein",
    "Value03": 2.0,
    "Value04": 1    // Layer 1 (z-index ordering)
}

Character Control

oncharacter: Displays a character. If the character is already on screen, it updates the image/position.

{
    "Index": 130,
    "type": "oncharacter",
    "Value01": "jung",      // Maps to character ID
    "Value02": "normal",    // Becomes assets/images/character/jung_normal.png
    "Value03": "center",    // Or "left", "right", "50%"
    "Value04": "fadein",
    "Value05": 1.0
}

anieffect: Applies a CSS animation class (like 'shake') to the character.

{
    "Index": 135,
    "type": "anieffect",
    "Value01": "jung",
    "Value02": "shake",     // Defined in global CSS
    "Value03": 0.5          // Duration
}

Dialogue

speak: The core storytelling element. Shows text and plays voiceover.

{
    "Index": 140,
    "type": "speak",
    "Value01": "jung",
    "Value02": "정성훈",      // Name shown
    "Value03": "This is a [FF0000]warning[-] message./nNew line here.",
    "Value04": "assets/voice/jung_01.mp3",
    "Value05": "#C5A059"    // Custom name color
}
Text Tags:
  • /n or \n: New Line
  • [RRGGBB]Text[-]: Color text (Hex code)

Interactions (Choice & Quiz)

choice: Presents 2 distinct options to the player.

{
    "Index": 200,
    "type": "choice",
    "Value01": "Enter the room",  // Option 1
    "Value02": 1000,              // Jump to Index 1000
    "Value03": "Run away",        // Option 2
    "Value04": 2000               // Jump to Index 2000
}

quiz01: Text input field for solving puzzles.

{
    "Index": 300,
    "type": "quiz01",
    "Value01": "Who is the culprit?",
    "Value02": ["Butler", "The Butler"], // Acceptable answers
    "Value03": "He wears a suit",        // Hint 1
    "Value04": "He was in the kitchen"   // Hint 2
}

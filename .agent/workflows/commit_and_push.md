1. Check the current git status
// turbo
2. Stage all changes
    ```bash
    git add .
    ```
3. Commit changes (Prompt user for message if not provided)
    ```bash
    git commit -m "Update: Apply recent changes"
    ```
    (Note: If you have a specific message, replace the string above)
4. Push to remote
    ```bash
    git push origin main
    ```
